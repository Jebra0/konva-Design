## About The Project

****_ This project is a web-based design tool built using Vue.js for the frontend and Laravel for the backend. The tool includes functionalities such as an image editor, text editor, and layers hierarchy, making it a versatile application for graphic design tasks.****_

### Prerequisites

- **Node.js**
- **npm or yarn**
- **PHP**
- **Composer**
- **MySQL**
